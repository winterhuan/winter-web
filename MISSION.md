# Mission: LLM

## Why

我要理解 LLM 的模型原理，用它写出可信、清晰的技术博客，并把这些理解落到 Agent 系统和 AI 应用产品设计中。

## Success looks like

- 能读懂《Attention Is All You Need》的核心论证，并用自己的话讲清 Transformer 为什么重要。
- 能把 LLM 原理写成面向工程读者的博客，而不是只复述术语。
- 能基于模型能力和限制，设计更可靠的 Agent 产品。

## Constraints

- 学习者会编程，懂一点深度学习。
- 学习者用过 OpenAI 和 Claude API，也做过 Agent。
- 课程应优先连接原理、写作和产品设计，不走纯数学推导路线。
- 原论文是英文，阅读门槛较高；课程应提供中文伴读、关键词拆解和博客化改写练习。

## Out of scope

- 暂不深入训练大型模型的工程基础设施。
- 暂不追求从零实现完整 Transformer 训练系统。
- 暂不把重点放在模型微调和分布式训练优化上。
