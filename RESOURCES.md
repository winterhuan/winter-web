# LLM Resources

## Knowledge

- [Paper: _Attention Is All You Need_ — Vaswani et al.](https://arxiv.org/abs/1706.03762)
  Transformer 原论文。Use for: Transformer 的问题设定、架构动机、self-attention、multi-head attention、position-wise feed-forward networks、positional encoding。
- [HTML: _Attention Is All You Need_ — arXiv HTML](https://arxiv.org/html/1706.03762v7)
  原论文的 HTML 版本。Use for: 快速引用原文段落和章节标题。
- [Guide: _The Illustrated Transformer_ — Jay Alammar](https://jalammar.github.io/illustrated-transformer/)
  高质量可视化解释。Use for: 建立 encoder/decoder、attention、Q/K/V 的直觉。
- [Slides: Stanford CS224N Lecture 8: Self-Attention and Transformers](https://web.stanford.edu/class/archive/cs/cs224n/cs224n.1244/slides/cs224n-2024-lecture08-transformers.pdf)
  Stanford NLP 课程讲义。Use for: 从 RNN 到 attention 的教学脉络，以及 Transformer 的课程级解释。
- [Notes: Stanford CS224N Self-Attention & Transformers](https://web.stanford.edu/class/cs224n/readings/cs224n-self-attention-transformers-2023_draft.pdf)
  Stanford 阅读笔记。Use for: 需要比博客更严谨、比论文更可读的解释时。
- [Article: _LLM Powered Autonomous Agents_ — Lilian Weng](https://lilianweng.github.io/posts/2023-06-23-agent/)
  Agent 系统综述。Use for: planning、memory、tool use 等 Agent 设计概念。
- [Guide: _A practical guide to building agents_ — OpenAI](https://openai.com/business/guides-and-resources/a-practical-guide-to-building-ai-agents/)
  面向产品和工程团队的 Agent 构建指南。Use for: 把 Agent 概念落到产品用例、工具、指令和评估。

## Wisdom (Communities)

- [Hacker News: Machine learning discussions](https://news.ycombinator.com/)
  工程视角强，适合测试博客选题和产品判断。Use for: 观察工程师对 LLM/Agent 论点的质疑点。
- [Reddit: r/LocalLLaMA](https://www.reddit.com/r/LocalLLaMA/)
  偏实践的开源模型社区。Use for: 观察模型能力、部署限制和工具链实践。
