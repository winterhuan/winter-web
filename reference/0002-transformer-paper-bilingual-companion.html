Reference 0002

Transformer 英文论文伴读

目标不是把论文逐字翻译成中文，而是让你能独立读懂关键句，并把它改写成适合博客的解释。

配合阅读： 《Attention Is All You Need》 的 Abstract 和 Introduction。先看本页，再回到原文。

四步读法

  1. 先找主干：谁做了什么，不急着翻每个修饰词。
  2. 圈术语块：sequence transduction、recurrent、convolutional、attention mechanism 这类词整块理解。
  3. 转成工程问题：旧方法哪里慢、哪里难并行、哪里不适合长距离依赖。
  4. 改成博客句：用中文讲给工程读者听，不追求论文腔。

Abstract 第一层：论文在反对什么

原文开头的大意是：当时最强的序列到序列模型，主要依赖复杂的 RNN 或 CNN，并且通常还会接 attention。

dominant models 当时主流、效果最好的模型。这里不是泛指所有模型。
sequence transduction 输入一个序列，输出另一个序列。机器翻译是典型例子。
recurrent or convolutional 论文要替换掉的旧架构路线：一步步读，或局部窗口扫。

博客化表达：过去做机器翻译，主流路线是用 RNN 或 CNN 处理序列，再用 attention 帮模型对齐输入和输出。Transformer 的出发点，是问一个更激进的问题：这些旧结构能不能都不要？

Abstract 第二层：论文提出什么

论文的核心主张可以压成一句话：Transformer 是一个只基于 attention 的新架构，完全去掉 recurrence 和 convolution。

new simple architecture 不是加一个小模块，而是换掉整体架构。
based solely on attention 只靠 attention 做序列内部的信息混合。
dispensing with 抛弃、去掉。这是 Abstract 里的强动作。

博客化表达：Transformer 的大胆之处不是“用了 attention”，而是把 attention 从辅助模块提升为主干结构。

最小英文词表

  • dominant：主流的、占优势的。
  • transduction：转换；在这里是序列到序列的转换。
  • configuration：配置、结构组合。
  • mechanism：机制，通常指一个可复用的计算模块。
  • solely：仅仅、只靠。
  • dispense with：去掉、摆脱。

练习

不看上面内容，把这句话补完整：

Transformer 的核心贡献不是 _____，而是 _____。

建议答案：不是“第一次使用 attention”，而是“只用 attention 替代 RNN/CNN 这类旧主干结构”。
