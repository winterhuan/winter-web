欢迎来到我的新网站。这是第一篇文章，用来测试排版，也顺便聊聊这个站点本身。

## 为什么自己搭网站

市面上的平台很多，但自己搭一个站点意味着：

- **完全的内容所有权** —— 文章、数据、样式都在你手里
- **想要的样子** —— 不受平台模板限制
- **极快的速度** —— 静态站点几乎瞬间加载

## 技术选型

这个站点用 [Astro](https://astro.build) 构建。它最大的特点是「内容优先」：

```js
// 内容集合让每篇文章都有类型保障
const post = defineCollection({
  schema: z.object({ title: z.string() }),
});
```

写文章只要丢一个 Markdown 文件到对应目录，自动就会出现在列表里。
