一个轻量的 Markdown 转换工具，左侧写 Markdown，右侧实时显示渲染结果。

## 特性

- 支持完整的 GitHub Flavored Markdown
- 实时预览，无需点击按钮
- 一键复制生成的 HTML
- 纯前端，无服务端，数据不离开浏览器

## 使用场景

- 写邮件模板时快速转换格式
- 给不支持 Markdown 的平台准备内容
- 学习 Markdown 语法
