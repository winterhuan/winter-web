Lesson 0003

Q/K/V：self-attention 的最小直觉

Transformer 能让 attention 接管模型主干，关键是 self-attention：每个 token 都能根据当前需要，主动读取其他 token 的信息。

本课主源： 《Attention Is All You Need》 第 3.2 节。先看 Q/K/V 论文伴读，再读原文。辅助阅读： The Illustrated Transformer。

先不要算矩阵

读公式前，先把 Q/K/V 当成三种“角色”。一个 token 不是只有一份表示；在 attention 里，它会被投影成 query、key、value 三种向量，用来完成一次信息检索。

Query 我现在想找什么信息。
Key 我这里有什么可被匹配的线索。
Value 如果你关注我，我真正交给你的内容。

一句话例子

句子是：The animal didn't cross the street because it was tired. 当模型处理 it 时，it 的 query 会去匹配其他词的 key；如果 animal 的 key 更匹配，模型就更多读取 animal 的 value。

这不是说模型真的“懂代词”。这里只是在建立机制直觉：self-attention 给每个 token 一次按相关性读取上下文的机会。

和旧主干的差别

RNN 的信息常常要沿时间步传过去；CNN 的信息常常要靠局部窗口和多层传播。self-attention 的直觉优势是：任意两个 token 可以直接计算相关性，所以长距离关系的路径更短，也更容易并行。
