# Triage 标签

工程技能使用五个标准 triage 角色。本文件把这些角色映射到本仓库问题追踪器中的实际标签名。

| mattpocock/skills 中的标签 | 本仓库使用的标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求已经明确，可以交给 AFK agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个 triage 角色时，使用上表右侧对应的标签名。

如果仓库以后改用不同的标签命名，只需要更新上表右侧列。
